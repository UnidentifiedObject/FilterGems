# FilterGems
A command-line tool that searches Modrinth for Minecraft mods within specific download ranges and mod loader categories.

## 🚀 The Problem This Solves
On the Modrinth website, you can usually only sort by "Most Downloads." This makes it hard to find "Hidden Gems"—stable, high-quality mods (e.g., 5,000–50,000 downloads) that are buried by viral hits.

## 🛠️ Installation & Usage
1. Download this repository.
2. Run `pip install -r requirements.txt` in your terminal.
3. **Windows Users:** Simply double-click `RunFilterGems.bat`.
4. **Other Users:** Run `python gem_hunter.py`.

# FilterGems

Belirli indirme aralıkları ve mod yükleyici kategorileri içinde Modrinth üzerinde Minecraft modları arayan bir komut satırı aracıdır.

## 🚀 Çözdüğü Sorun
Modrinth web sitesinde genellikle sadece "En Çok İndirilenler"e göre sıralama yapabilirsiniz. Bu durum, 5.000 ile 50.000 indirme arasındaki kaliteli "Gizli Cevherleri" bulmayı zorlaştırır.

## 🛠️ Kurulum ve Kullanım
1. Bu depoyu indirin.
2. Terminalinizde `pip install -r requirements.txt` komutunu çalıştırın.
3. **Windows Kullanıcıları:** `RunFilterGems.bat` dosyasına çift tıklayarak çalıştırabilir.
4. **Diğer Kullanıcılar:** `python gem_hunter.py` komutunu kullanabilir.
